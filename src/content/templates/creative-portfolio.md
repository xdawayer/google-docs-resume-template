**Creative Portfolio** is for design-led roles where the resume is part of the portfolio: a
teal accent, an optional avatar, and skill meters that read as confident without being loud.
It is built for a PDF a person will actually look at.

Open it in the builder, add a local photo if you like, tune the accent color, and export a
selectable-text PDF. For applications that screen with strict ATS software first, pair it with
a single-column template like Modern ATS.
